# hermes-life-os

Personal operating system agent that runs inside Hermes. Detects your daily patterns, learns your routines over time, and helps you stay organized — all using Hermes's persistent memory system.

## Features

- Daily pattern detection and routine learning
- Habit tracking and streak monitoring
- Time management suggestions
- Persistent lifestyle context across sessions
- Weekly summaries and insights

## Installation

```bash
git clone https://github.com/Lethe044/hermes-life-os.git ~/.hermes/skills/hermes-life-os
```

## Usage

```
/skill hermes-life-os
/life-os status       # Today's overview
/life-os log "did X"  # Log an activity
/life-os insights     # Weekly pattern analysis
```
